# Peramalan Saham dengan Model GJR-GARCH

## Deskripsi
Proyek ini bertujuan melakukan peramalan saham menggunakan model Glosten-Jagannathan-Runkle Generalized Autoregressive Conditional Heteroskedasticity (GJR-GARCH).

## Tujuan
- Menganalisis karakteristik volatilitas return saham
- Menerapkan model GJR-GARCH untuk peramalan volatilitas

## Metodologi
- Data: Return saham historis periode multi-tahun
- Pendekatan: Pemodelan volatilitas time series
- Model: GJR-GARCH (untuk data asimetris)
- Evaluasi: Pengukuran akurasi model peramalan

## Keunggulan Pendekatan
- Mampu menangkap efek asimetris pada volatilitas return
- Mendukung perencanaan investasi jangka panjang

## Manfaat
- Menyediakan dasar analisis volatilitas berbasis statistik
- 
## Kesimpulan
Pemodelan volatilitas return saham menggunakan pendekatan GJR-GARCH efektif untuk menganalisis fluktuasi dan risiko investasi serta dapat dimanfaatkan sebagai alat bantu dalam perencanaan dan pengambilan keputusan investasi berbasis data.
